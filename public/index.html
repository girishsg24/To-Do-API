SJSU Team 5's To-Do API

We make your life easier, Create and manage your tasks in a Jizzy!

To Create a task

Use postman and send a post request to the url https://sjsu-team5-todo-api.herokuapp.com/todos/ with a JSON object.

JSON object format Eg:

{

"description" : "Be Batman and save the city!",

"completed" : false

}

To Update a task

Use postman and send a put request to the url https://sjsu-team5-todo-api.herokuapp.com/todos/'id' with a JSON object.

'id' being equal to the id of the task that you would like to update.

JSON object is again in the same format as above but you can decide to skip the attribute that you do not intend to change.

To Delete a task

Use Postman and send a delete request to the url https://sjsu-team5-todo-api.herokuapp.com/todos/'id'

'id' being equal to the id of the task that you would like to delete.

To retrieve all the tasks

Use Postman and send a get request to the url https://sjsu-team5-todo-api.herokuapp.com/todos

To retrieve a selected task.

Use Postman and send a get request to the url https://sjsu-team5-todo-api.herokuapp.com/todos/'id'

'id' being equal to the id of the task that you would like to fetch.

To retrieve task based on filters.

Use Postman and send a get request to the url https://sjsu-team5-todo-api.herokuapp.com/todos followed by ? and the query.

For Eg: To fetch all the tasks with completed status set to false

https://sjsu-team5-todo-api.herokuapp.com/todos/?completed=false

Similarly to fetch all the taks whose description contains a certain word.

https://sjsu-team5-todo-api.herokuapp.com/todos/?q=batman

You could decide to chain to have both the filters.

https://sjsu-team5-todo-api.herokuapp.com/todos/?q=batman&completed=false

Thank You!
